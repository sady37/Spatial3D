# Spatial3D
项目定位与核心目标：本项目是养老跌倒监测系统 V3.0 的核心组成部分，旨在利用固定安装的 TI AWR6844AOP 毫米波雷达在房间无人或静止期间进行长期累积扫描，建立一套可作为后续跌倒检测空间先验、可持续更新的高分辨率 3D 静态分层空间建模系统。

系统架构与技术路线：受限于雷达无法输出 RAW I/Q 数据及 UART 带宽，系统放弃实时建图，采用“DSP 端实时计算并维护体素统计信息、UART 周期性同步体素地图、PC 端执行语义建模与墙体拟合”的三层计算架构。

工程价值与应用场景：项目通过长期积分策略攻克了硬件资源受限的瓶颈，最终输出包含墙体、家具等语义信息的持久化 3D 静态背景地图，可直接作为后续各类 AI 模型、论文、PRD 或设计文档的统一项目背景。
